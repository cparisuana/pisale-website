About the Code

A robust framework for complex hydrodynamic simulations

High Performance

Optimized algorithms for efficient computation of large-scale problems with parallel processing capabilities

Adaptive Methods

ALE framework seamlessly transitions between Lagrangian and Eulerian descriptions for optimal accuracy

Smart Refinement

AMR technology dynamically refines mesh where needed, balancing resolution and computational cost

Key Features

Cutting-edge capabilities for modern CFD challenges

01

ALE Formulation

Arbitrary Lagrangian-Eulerian framework handles extreme deformations while maintaining numerical accuracy

  • Automatic rezoning
  • Contact algorithms
  • Material interface tracking
02

Adaptive Mesh Refinement

Dynamic grid adaptation concentrates resolution in critical regions automatically

  • Error-driven refinement
  • Multi-level hierarchy
  • Load balancing
03

2D & 3D Capabilities

Full support for both 2D and 3D simulations with consistent numerical schemes

  • Planar and axisymmetric 2D
  • Full 3D geometries
  • Coupled multi-physics
04

Advanced Physics

Comprehensive physical models for realistic simulations

  • Equation of state library
  • Strength models
  • Fracture mechanics
